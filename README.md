# MassageTime (iOS)

MassageTime is a simple iOS app built with SwiftUI for managing massage appointments.
The application is designed for massage therapists who want a clear and lightweight way
to manage clients and bookings directly on the device.

The project was created as an educational SwiftUI application and focuses on clean structure,
clarity, and real-world iOS development practices.

---

## About the App

MassageTime allows a massage therapist to:

- manage clients
- create and view bookings
- see appointment details
- review client appointment history

All data is stored locally on the device.
The app does not use a backend, login system, or external services.

---

## Features

- Home screen with today’s bookings
- Create new bookings
- Client list with individual client profiles
- Appointment history per client
- Localization-ready (English / Swedish)

---

## Project Context

This application was developed as a school project and prepared for App Store submission.
The goal of the project is to demonstrate a complete iOS app workflow, including
project structure, SwiftUI implementation, privacy handling, and App Store requirements.
